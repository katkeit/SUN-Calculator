❈ Selecting a class

To start a character the user must select a class. Scroll over to the drop down box in the "Class Selection" section and click on the down arrow. A drop down menu will appear and now the user can select an available class.

The subsection "Additional Stats" inside the main section "Character Information" will update to show the base stats for the selected character as shown below.

❈ Edit level

Before the user can begin editing the stats, the user must edit the level inside box in the "Level" section. The user can click on the box, press 'Backspace' to remove the default '1', type any number from 1 to 195, and then press 'Enter'. Pressing 'Enter' will adjust the information inside the "Extra Points" and "Additional Stats" sections as show below.

❈ Edit stats

The user can edit the stats inside the section "Stats" underneath the section "Level" by clicking inside the box to the right of the corresponding stat. From there, the user can 'Backspace' inside the box to remove any preset numbers, type in the numbers of their choice, and press 'Enter'. Pressing 'Enter' will automatically update the information in the "Additional Stats" section as shown below (Number 1 and 2).
